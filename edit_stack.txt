스택(Stack)은 마지막에 들어간 데이터가 가장 먼저 나오는 후입선출(LIFO, Last-In-First-Out) 구조를 가진 자료구조로, 코딩에서 다양한 핵심 기능에 사용됩니다. 
주요 사용 예시는 다음과 같습니다.
함수 호출 관리 (Function Call Stack): 프로그램에서 함수가 호출될 때, 현재 실행 중인 함수의 상태를 스택에 저장(Push)하고, 함수가 종료되면 스택에서 꺼내어(Pop) 이전 위치로 돌아갑니다.
웹 브라우저의 '뒤로 가기/앞으로 가기': 사용자가 방문한 페이지들을 스택에 쌓고, 뒤로 가기를 누르면 가장 최근 방문한 페이지가 나오는 방식으로 작동합니다.
실행 취소(Undo) 및 다시 실행(Redo): 워드 프로세서나 포토샵 같은 애플리케이션에서 사용자의 동작을 스택에 저장하여, Ctrl+Z를 누르면 가장 최근의 작업을 취소하는 데 사용됩니다.
괄호 유효성 검사 (Syntax Parsing): 컴파일러나 에디터가 소스 코드 내의 괄호((), {}, [])가 올바르게 열리고 닫혔는지 확인할 때 스택을 활용합니다.
깊이 우선 탐색 (DFS, Depth-First Search): 그래프나 트리 구조에서 한 방향으로 끝까지 탐색하다가 막히면 이전 갈림길로 돌아가서 다른 방향을 탐색하는 알고리즘에 스택이 사용됩니다.
수식 계산 (후위 표기법): 계산기 프로그램에서 연산 우선순위를 처리하기 위해 중위 표기식(1+23)을 후위 표기식(123+)으로 변환하거나 계산할 때 사용됩니다. 
코딩 예시 (Python):
python
stack = []
# 데이터 추가 (Push)
stack.append(1)
stack.append(2)
stack.append(3) # stack: [1, 2, 3]

# 데이터 삭제 (Pop) - 가장 최근인 3이 나옴
print(stack.pop()) # 출력: 3
print(stack)       # 출력: [1, 2]
