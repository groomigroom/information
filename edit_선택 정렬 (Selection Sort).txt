선택 정렬(Selection Sort)은 데이터가 정렬되지 않은 부분에서 최솟값(또는 최댓값)을 찾아 정렬되지 않은 부분의 첫 번째 원소와 교체하는 방식으로 작동하는 간단한 정렬 알고리즘입니다.

1. 코딩에서의 사용 예시
예시: 파이썬(Python)으로 구현한 선택 정렬
python
def selection_sort(arr):
    for i in range(len(arr) - 1):
        # 1. 정렬되지 않은 부분에서 최솟값의 인덱스를 찾음
        min_index = i
        for j in range(i + 1, len(arr)):
            if arr[j] < arr[min_index]:
                min_index = j
        
        # 2. 최솟값을 정렬되지 않은 부분의 첫 번째 원소와 교체
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

# 사용 예시
data = [64, 25, 12, 22, 11]
sorted_data = selection_sort(data)
print(sorted_data)  # 출력: [11, 12, 22, 25, 64]
2. 선택 정렬이 유리한 경우
데이터의 개수가 적을 때: 



의 시간 복잡도를 가지므로 대규모 데이터에는 부적합하지만, 데이터 양이 적으면 매우 효율적입니다.
교환(Swap) 비용이 클 때: 다른 알고리즘에 비해 데이터 위치를 바꾸는 횟수(최대 
번)가 적기 때문에, 데이터를 메모리에서 옮기는 비용이 비싼 경우에 유리할 수 있습니다.
제자리 정렬(In-place Sort)이 필요할 때: 별도의 추가적인 메모리 공간을 거의 사용하지 않고 입력 배열 내에서 정렬을 마쳐야 하는 경우에 적합합니다.
