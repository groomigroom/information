자료 사전
=

+

( ) 자료의 생략

[ | ] 자료의 선택 (또는) (입력은 파이프 \자에다 쉬프트 누르면 됩니다.)

{ } 자료의 반복
-> 
{n,} : n번 이상 반복 (최소 횟수 지정) 
의미: 앞의 문자가 최소 
번 이상 연속으로 나타나야 합니다.
최대 횟수: 제한이 없습니다 (
부터 무한대).
표기: 쉼표(,) 뒤에 숫자가 없습니다.
예시: a{3,} -> 'aaa', 'aaaa', 'aaaaa', ... (a가 3개 이상인 모든 경우) 

{,n} : 최대로 n번 반복 (최대 횟수 지정 - 확장된 표기법) 
의미: 앞의 문자가 0번부터 최대 
번까지 나타날 수 있습니다.
최소 횟수: 0번 (없어도 됨)
표기: 쉼표(,) 앞에 숫자가 없고 뒤에만 있습니다.
특징: GNU grep 등 일부 정규식 엔진에서 지원하는 확장 기능(Extension)입니다.
예시: a{,3} -> '', 'a', 'aa', 'aaa' (a가 3개 이하인 모든 경우) 

{m,n} : m번 이상 n번 이하 반복 (범위 지정) 
의미: 앞의 문자가 최소 
번 이상, 최대 
번 이하 반복됩니다.
예시: a{2,4} -> 'aa', 'aaa', 'aaaa' (a가 2개 이상 4개 이하)
